Hireflow is a modern web application that helps users discover and explore freelance and remote job opportunities. It provides powerful filtering, sorting, and search features to efficiently browse jobs and manage favorites.

🚀 Features
🔍 Search jobs by keyword (e.g., React, Python, Design)
📋 View job details (title, company, salary, tags)
🔽 Filter jobs by category and skills
🔃 Sort jobs by salary and latest postings
⭐ Save favorite jobs for later
🌐 Direct link to apply for jobs
💾 Persistent data using LocalStorage
🎨 Clean and responsive user interface

interface
🛠️ Tech Stack
Frontend: React.js
Styling: CSS
API: Remotive Jobs API
State Management: React Hooks
Storage: LocalStorage
⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/your-username/freelancehub.git
cd freelancehub

Install dependencies:

npm install

Start the application:

npm start

Remotive Jobs API
